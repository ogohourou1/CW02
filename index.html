    HTML tags classes that may be used by The school of Art and Science for the website

  • body
    • The body can have attributes such as a background color for the website
  • h1, h2, h3, ...
    • The headers can define classes based on their purpose. For examples a header for the course name. Useful attributes for head can be the font, color, or style
  • a
  • link
  • ul
  • ol
  • table
    • Tables can use in-line styling to make them coherant on the website for example alignments and background image or color
    • Tables have sub tags like tr and td which can use attributes like rowspan or colspan to customize the tables
  • style
    • Teachers can use embedded styles to add further customization to their respective website which will span over the whole website
    • In-line styling is also useful to make direct changes to selectors in case teachers need to over-ride the external or embedded stylesheet
  • p
    • Paragraphs can receive atttributes like borders to dinstiguish them
    • Paragraphs also can have attributes which change the text color or change the background color.
